General tips
  • If you find any bugs or if you have any suggestions please report here.
    For questions the application developer and system administrator is found at josebraga@fct.unl.pt.
  • This is a mobile-ready website.
  • In case you forget your password please ask a manager to generate a new one.
  • There are several user permission levels:
    • Standard: The level attributed to the majority of users. These users are able to read and change data in "Myself" (except affiliation data). They have no access to any other features.
    • Team: For users managing their team data (members and publications). These users are able to pre-register new members.
    • Local Managers: Users that manage data from all teams within a location (e.g. Lisboa, Porto). They are able to read data from all users but can only change data from users with the same location as them.
    • Global Managers: Users that manage data from all teams anywhere.
